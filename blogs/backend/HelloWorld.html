    Flask入门笔记

    FeliksLv 2019/8/8 flask

    # 环境搭建

    # 1. 安装pipenv

    sudo -H pip install pipenv
    1

    # 2. 创建虚拟环境

    新建一个文件，在文件夹里打开终端

    pipenv install
    1

    激活虚拟环境

    pipenv shell
    1

    vscode设置 在项目中的.vscode文件夹中创建settings.json文件。

    # 获取当前虚拟环境的地址
pipenv --venv
    1
    2

    在settings.json中加入下面一行设置，里面填上你自己的虚拟环境的地址。

    "python.venvPath": "..."，
"python.linting.pylintArgs": [
    "--load-plugins",
    "pylint-flask"
],
"python.formatting.autopep8Path": "autopep8",
"python.linting.pylintPath": "pylint"

    # 3. 安装flask

    设置镜像(修改Pipfile)

    url = "https://pypi.tuna.tsinghua.edu.cn/simple/"
    1
    pipenv install flask
    1

    # 4. 启动

    创建app.py

    from flask import Flask

app = Flask(__name__)

@app.route('/')
def index():
    return '<h1>Hello Flask!</h1>'

if __name__ == "__main__":
    app.run(host='0.0.0.0', port=4555, debug=True)

    pipenv install python-dotenv
    1

    使用命令运行即可

    flask run
